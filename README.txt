# Hanoi-app
This is a simple application for solving the Tower of Hanoi problem. The application uses a recursive algorithm to calculate the steps required to move discs and provides a user-friendly frontend interface to display the solving process.

## Features
- Accepts user input for the number of discs.
- Calculates and displays the step-by-step process of moving discs using a recursive algorithm.
- Provides an intuitive frontend interface to visualize each step of the process.

## System Requirements
- Node.js: v18.17.0
- React: 18.2.0
- npm: 9.6.7

## Installation and Running
1. Ensure that port 3000 is not already occupied by another application.
2. Clone this repository to your local environment.
3. Install the necessary dependencies by running the following command:
npm install
4. Run the provided test suite to verify the functionality:
npm test


## Usage

1. Open your preferred web browser and navigate to http://localhost:3000/.
2. Enter your desired number of discs directly into the frontend interface and press the "Enter" key.
